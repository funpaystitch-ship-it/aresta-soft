## ⚡ Установка и запуск (1 команда)

### CMD (Командная строка)
```cmd
git clone https://github.com/funpaystitch-ship-it/aresta-soft && cd aresta-soft && python aresta_soft.py
```

### PowerShell
```powershell
git clone https://github.com/funpaystitch-ship-it/aresta-soft; cd aresta-soft; python aresta_soft.py
```
